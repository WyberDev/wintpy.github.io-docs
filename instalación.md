# Instalación de WintPy
Actualmente WintPy solo esta soportado para Linux, pero su instalación es facil, puedes clonar el repositorio con:
```git clone https://github.com/WyberDev/WintPy.git```
O descargar la ultima release en Github desde el navegador

Cuando está instalado en tu carpeta, posee un archivo llamado `ejecutar.sh` que puedes usarlo escribiendo en tu terminal `./ejecutar.sh` o incluso ejecutarlo escribiendo ```python WintPy.py``` en tu terminal.

# Al ejecutarlo
Al ejecutar WintPy te hará escribir un nombre para hacerte un usuario, puedes omitirlo escribiendo 'Q', o incluso omitirlo cada inicio con 'S'
