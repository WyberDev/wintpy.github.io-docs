Comandos de WintPy

La lógica de comandos en WintPy a diferencia de cualquier lenguaje de scripting es que solo tienes que escribir el comando y al presionar enter te deja ingresar el siguiente operando
Aquí hay un ejemplo:


  cd
  Ir a directorio:

En WintPy existen varios comandos, esta es la explicación de todos ellos:

  • --help = Este comando muestra ventana indicando para que sirven todos los comandos

    • ───────── Navegación y manejo de archivos ───────────

  • mkdir = Es la abreviación de "Make Directory", y como el nombre lo dice,
    te deja crear un directorio/carpeta

  • mkfle = Es la abreviación de "Make File" Este comando introduce una funcionalidad nunca antes vista, porque mkfle te permite crear archivos con su propio formato
    y de una manera muy simple.

  • rmdir = Es la abreviación de "Remove Directory" Te deja eliminar un directorio/carpeta

  • rm = Es la abreviación de "Remove" Te deja eliminar archivos.

  • cd = Permite ir a un directorio y operar desde ese mismo directorio.
    Tiene una variante llamada cd .. que es para retrodecer un directorio atrás.

  • cp = Es la abreviación de "Copy" Te deja copiar un archivo, no funciona
    con directorios.

  • pt = Es la abreviación de "Paste" Te deja pegar un archivo que hayas
    copiado anteriormente con cp al pegar algo te preguntara con que nombre
    quieres pegarlo. Puedes escribir 'same' para reutilizar el mismo nombre.

  • ls = Este comando te permite mostrar los archivos y directorios en los
    que estás operando.

  • edit = Te deja seleccionar un archivo para editarlo, con un editor
    de texto propio de WintPy llamado PyEditor.

    • ───────────────── Misceláneo ───────────────────

  • exit = Te permite salir de WintPy.

  • clear = Limpia todo el texto en pantalla para una mejor legibilidad.

  • refetch = Muestra información de WintPy o otras cosas, como tu sistema
    operativo real.

  • logout = Te deja cerrar sesión. Durante la sesión cerrada no se puede
    ejecutar ningún comando que no sea login, que sirve para iniciar sesión.

  • devmode = Activa una serie de comandos altamente recomendados para desarrolladores. Puedes aprender más de esto en 2.1. Comandos de desarrollador

  • mkuser = Te permite crear usuarios, te preguntara si quieres integrar carpetas
    de paquetes, con esto se refiere a la carpeta 'packages' que te permite tener tus propias aplicaciones en un solo usuario.

    • ───────────────── Aplicaciones ───────────────────

  • pkg -list = Te muestra el listado de aplicaciones disponibles de WintPy y te permite
    descargarlas.

  • pkg -remove = Te permite eliminar aplicaciones que tengas instaladas.

  • exec = Te permite ejecutar una aplicacion que tengas instalada.

    • ───────────────── Sandboxing ───────────────────

  • disa = WintPy posee un sistema de seguridad llamado 'Sandbox' que bloquea la interacción con el sistema por seguridad, con este comando vas a poder desactivarlo. Pero no está recomendado.

  • ensa = Permite reactivar el Sandbox si lo desactivaste previamente.
