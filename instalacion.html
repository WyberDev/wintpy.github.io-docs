Instalación de WintPy

Actualmente WintPy solo tiene soporte para Linux y Android, su instalación
es muy simple, puedes clonar el repositorio ejecutando
git clone https://github.com/WyberDev/WintPy.git en tu terminal o descargar
la última release en Github desde tu navegador preferido

El único requisito de WintPy es tener Python 3.0+

Al ejecutarlo

Al ejecutar WintPy te hará escribir un nombre para hacerte un usuario, puedes omitirlo escribiendo 'Q', o incluso omitirlo cada inicio con 'S'
