Comandos de desarrollador

Cuando activas el modo desarrollador con devmode desbloqueas varios
comandos aptos para el desarrollo de aplicaciones y otras cosas. Estos son todos:

  • -d = Sirve como una ventana de ayuda para mostrarte que hacen todos
    los comandos de desarrollador.

  • autojson = Este comando es demasiado útil para la creación de aplicaciones:
    Cada aplicación en WintPy requiere de un archivo llamado 'info.json' para identificar la aplicación, autojson te permite autogenerar esos archivos, su interfaz se ve así:
